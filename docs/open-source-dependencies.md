# Open-source dependencies

ScanLeno keeps documents on-device by default. No package in this list uploads
user files to a server.

| Package | License | Purpose |
| --- | --- | --- |
| camera | BSD-3-Clause | In-app camera preview and manual capture. |
| image_picker | BSD-3-Clause | Single and multi-image gallery selection. |
| image | MIT | On-device page rotation and image processing. |
| path_provider | BSD-3-Clause | Application-private document storage. |
| pdf | Apache-2.0 | Offline PDF generation. |
| share_plus | BSD-3-Clause | Native system sharing sheet. |

Licenses must be rechecked whenever a dependency is upgraded.
